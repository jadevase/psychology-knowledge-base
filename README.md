# 心理咨询师考试知识库

> 🧠 系统化备考资源 | 180 天学习路径 | 每日精进稳步通关

## 📖 项目简介

本知识库是为准备**心理咨询师考试**的学习者打造的系统性学习资源库，包含完整的 180 天学习计划、每日详细课程内容、配套习题练习和知识点索引。

### ✨ 核心特色

- **科学规划**：6 个月（180 天）学习周期，分为六个递进阶段
- **每日任务**：每天 50 分钟自学 + 10 分钟习题，高效利用时间
- **自包含内容**：所有专业名词均有详尽讲解，无需额外参考书籍
- **循序渐进**：从基础理论到实践技能，全面覆盖考试大纲
- **在线访问**：通过 GitHub 原生渲染，随时随地在线阅读

## 📚 学习内容

### 六大学习阶段

| 阶段 | 主题 | 天数 | 核心内容 |
|------|------|------|----------|
| 第一阶段 | 基础心理学 | 第 1-30 天 | 心理学的研究对象、方法、心理过程、个性心理 |
| 第二阶段 | 发展心理学 | 第 31-60 天 | 毕生发展观、婴幼儿期、青少年期、成年期发展 |
| 第三阶段 | 社会心理学 | 第 61-90 天 | 社会化、社会认知、人际关系、群体心理 |
| 第四阶段 | 变态心理学 | 第 91-120 天 | 心理障碍分类、诊断标准、常见异常行为 |
| 第五阶段 | 咨询心理学 | 第 121-150 天 | 咨询理论、技术、伦理、案例分析 |
| 第六阶段 | 操作技能 | 第 151-180 天 | 实践应用、模拟考试、答题技巧 |

## 🚀 快速开始

### 方式一：在线阅读（推荐）

1. 访问 [index.html](https://github.com/jadevase/psychology-knowledge-base/blob/main/index.html) 导航页面
2. 点击"每日课程"进入当天学习内容
3. 完成 50 分钟阅读后，返回点击"习题练习"进行测试

### 方式二：本地学习

```bash
# 克隆仓库到本地
git clone https://github.com/jadevase/psychology-knowledge-base.git

# 进入项目目录
cd psychology-knowledge-base

# 用 Markdown 阅读器打开 daily-lessons/day-001-full.md
```

## 💡 使用建议

### 最佳学习流程

1. **预习**（5 分钟）：浏览当日课程的标题和小结，了解学习框架
2. **精读**（35 分钟）：逐节阅读详细内容，理解每个概念的定义
3. **复习**（10 分钟）：回顾重点概念和易错点提醒
4. **练习**（10 分钟）：完成配套习题，检验学习效果
5. **总结**（5 分钟）：记录学习笔记和疑问

### 时间管理

- ⏰ 建议固定每天同一时间段学习（如下午 5:00）
- 📱 学习时关闭手机通知，保持专注
- ☕ 每 25 分钟休息 5 分钟，采用番茄工作法
- 📝 准备笔记本记录关键概念和个人思考
