# 心理学知识库使用指南

## 📚 知识库概述

这是一个为心理咨询师考试自学设计的知识库系统，具有以下特点：

- **自动同步**：每日学习内容自动添加到知识库
- **结构化组织**：按学习阶段和天数组织内容
- **配套练习**：每天配有 10 道练习题和答案解析
- **进度追踪**：通过索引文件记录学习进度

## 📁 目录结构

```
psychology-knowledge-base/
├── README.md                 # 项目说明
├── USER_GUIDE.md            # 使用指南（本文件）
├── index.json               # 知识库存索（学习进度、课程列表）
├── learning-plan.md         # 完整学习计划
├── content-format-spec.md   # 内容格式规范
├── daily-lessons/           # 每日学习内容
│   ├── day-template.md      # 学习内容模板
│   ├── day-001.md           # 第 1 天学习内容
│   └── ...
├── practice-questions/      # 练习题
│   ├── day-001-questions.md # 第 1 天练习题
│   └── ...
└── sync_simple.py           # 同步脚本
```

## 🔄 自动同步机制

### 定时任务配置

已配置两个定时任务：

1. **心理学每日学习推送**（每天 17:00）
   - 推送当日学习内容到钉钉
   - 包含 50 分钟阅读材料和 10 道习题

2. **心理学知识库同步**（每天 17:05）
   - 在学习推送后 5 分钟执行
   - 自动生成学习内容并保存到知识库
   - 更新索引文件和进度统计

### 同步流程

```
每天 17:00 → 学习推送到钉钉
    ↓
每天 17:05 → 知识库同步任务
    ↓
读取 index.json → 获取当前学习进度
    ↓
生成下一天的学习内容
    ↓
保存到 daily-lessons/day-XXX.md
    ↓
生成练习题
    ↓
保存到 practice-questions/day-XXX-questions.md
    ↓
更新 index.json 进度统计
```

## 📖 如何使用知识库

### 查看学习进度

打开 `index.json` 文件，查看：
- `current_day`：当前学习到第几天
- `total_lessons`：已生成的总课程数
- `stages`：各学习阶段的状态

### 查找特定内容

**按天数查找**：
- 学习内容：`daily-lessons/day-XXX.md`
- 练习题：`practice-questions/day-XXX-questions.md`

**按主题查找**：
使用文本编辑器搜索关键词，或查看 `index.json` 中的课程列表。

### 复习已学内容

1. 打开 `daily-lessons/` 目录
2. 按文件名排序（day-001, day-002...）
3. 选择需要复习的天数打开

### 做练习题

1. 打开对应的 `practice-questions/day-XXX-questions.md`
2. 先独立完成所有题目
3. 对照答案和解析
4. 记录错题，重点复习相关知识点

## 🛠️ 手动操作

### 手动触发同步

如果需要手动生成某一天的内容，可以运行：

```bash
cd psychology-knowledge-base
py sync_simple.py [天数]
```

例如，生成第 5 天的内容：
```bash
py sync_simple.py 5
```

### 初始化知识库

如果要重新生成前 7 天的示例内容：

```bash
cd psychology-knowledge-base
py init_knowledge_base.py
```

### 查看知识库状态

读取 `index.json` 文件：
```bash
cat index.json
```

或在 Python 中：
```python
import json
with open('index.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
    print(f"当前进度：第{data['current_day']}天")
    print(f"总课程数：{data['total_lessons']}")
```

## 📊 学习内容格式

每天的学习内容包括：

1. **学习目标**：3-5 个核心知识点
2. **核心概念**：详细讲解每个专业术语
3. **时间分配建议**：50 分钟学习规划
4. **重点提示**：标注重要性和记忆要点

练习题包括：

1. **10 道选择题**：覆盖当天核心知识点
2. **答案**：单独列出
3. **解析**：解释为什么选择这个答案

## 💡 使用建议

### 高效学习法

1. **预习**：快速浏览学习目标和核心概念标题（5 分钟）
2. **精读**：深入理解每个概念的详细讲解（30 分钟）
3. **总结**：回顾重点，做笔记（10 分钟）
4. **练习**：完成 10 道练习题（10 分钟）
5. **订正**：对照答案，理解错题（5 分钟）

### 复习策略

- **当日复习**：学习后 24 小时内快速回顾
- **周末复习**：每周日复习本周内容
- **阶段复习**：完成一个阶段后做综合练习

### 笔记整理

建议使用以下格式做笔记：

```markdown
# 第 X 天：[主题]

## 核心概念
- 概念 1：[自己的理解]
- 概念 2：[自己的理解]

## 易错点
- [记录容易混淆的点]

## 练习题错题
- 第 X 题：[错误原因分析]
```

## 🔧 故障排查

### 同步任务未执行

检查定时任务状态：
```bash
use_cron action=list
```

### 文件生成失败

检查脚本执行日志，确认：
- Python 环境正常
- 目录权限正确
- 磁盘空间充足

### 索引文件损坏

如果 `index.json` 损坏，可以：
1. 备份现有文件
2. 重新运行初始化脚本
3. 手动重建索引

## 📞 技术支持

如遇到问题，请检查：
1. 定时任务配置是否正确
2. 脚本文件是否完整
3. 目录结构是否符合规范
